# 行为准则

英文版：[CODE_OF_CONDUCT.md](CODE_OF_CONDUCT.md)

## 我们的标准

我们致力于建设一个尊重他人、无骚扰的社区环境。

积极行为示例：
- 保持尊重并提供建设性意见。
- 先假设他人出于善意。
- 提供可执行、可落地的反馈。

不可接受的行为包括：
- 骚扰、侮辱或歧视。
- 引战、人身攻击。
- 未经同意公开他人隐私信息。

## 执行

项目维护者可删除、编辑或拒绝违反本准则的评论、提交、Issue 或 PR。

## 适用范围

本准则适用于本项目的所有协作空间，包括 Issues、Pull Requests 和 Discussions。

## 举报

如遇到不当行为，请尽可能通过私下渠道向项目维护者举报。
